# 发票助手（PDF 批量提取 + GUI 导出）

一个 Python GUI 工具，用于批量读取发票 PDF，并提取以下字段导出为 `CSV` 或 `XLSX`：

- 发票号码
- 开票日期
- 购买方信息：名称、统一社会信用代码/纳税人识别号
- 销售方信息：名称、统一社会信用代码/纳税人识别号
- 项目名称、规格型号、单位、数量、单价、金额、税率/征收率、税额
- 合计

支持：
- 批量上传 PDF
- 从列表中移除/上移/下移（自定义处理顺序）
- 合并整理（一个总表）
- 分开整理（每个 PDF 一个表）
- 进度条显示处理状态
- XLSX 自动美化表头颜色与列宽

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python invoice_gui_extractor.py
```

## 说明

1. 若 PDF 是电子发票文本版，通常可直接提取。
2. 若 PDF 实际是扫描图（图片型），需要先 OCR，再处理。
3. 当前规则已按你提供的样例版式做适配；如后续有不同模板，可继续补充规则。

## 你怎么给我样例更方便？

如果你后续要我继续优化识别准确率，请按下面方式提供：

- **优先**：发 2~5 份脱敏后的真实 PDF（保持原版式）
- 或者：
  - 全图截图（整页）
  - 分辨率尽量高
  - 圈出识别错误字段
  - 告诉我“期望值”是什么

这样我可以快速加规则，让你的模板识别更稳。
